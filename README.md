# mianwrapper
面函数修复

# 用法
把[弄面.h](nomian.h)放进你的项目里并且在出现面函数的文件内include：
```C
#include "nomian.h"
  
(other part of C source...)
```
要注意的是面函数应当只出现一次，include弄面头文件也应该只出现一次
